# Attribution

The TimeZoner Raycast extension includes bundled timezone boundary data for its
static map preview.

Boundary data is derived from
[timezone-boundary-builder](https://github.com/evansiroky/timezone-boundary-builder),
which builds timezone boundaries from OpenStreetMap data. The upstream code is
MIT licensed. The boundary database is made available under the Open Database
License (ODbL) v1.0 and must keep OpenStreetMap attribution.

Required notice:

> Contains information from timezone-boundary-builder and OpenStreetMap, which
> is made available under the Open Database License (ODbL) v1.0.
